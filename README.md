# Admin Dashboard Panel

## Overview
The Admin Dashboard Panel is a web and Android-based application designed for administrators to manage and monitor the backend data, users, and system activities in a user-friendly interface. The project provides an intuitive admin panel to help manage different modules effectively and in real-time.

## Features
- **User Management**: Add, update, and delete users.
- **Analytics & Reports**: View charts, statistics, and reports.
- **Notifications**: Real-time notifications to admin.
- **Role-based Access Control**: Define roles and permissions for users.
- **Responsive UI**: Optimized for both web and mobile views.
- **Real-time Data Sync**: Update data instantly across both platforms.

## Tech Stack
- **Web**: React.js 
- **Android**: React native
- **Authentication**: JWT for secure login and session management
